Modo Interativo
Use o modo interativo do interpretador do Python para escrever e executar uma linha de código do Python de cada vez.
Às vezes, o modo interativo é chamado de "REPL", que é um acrônimo para "read-eval-print loop". Muitas linguagens de programação têm um REPL que pode ser usado para essa mesma finalidade.
